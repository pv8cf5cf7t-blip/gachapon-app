# 专业手相分析标准 (Professional Palmistry Standards)

> 创建于 2026-05-11，用户要求："再专业一点" → 制定此标准

## 核心原则

1. **禁止娱乐型泛泛解读** — 每句话必须有手相学依据
2. **古典术语强制使用** — 禁用俗称，用专业术语
3. **量化一切** — 掌丘用★评级，性格用雷达图 0-10 分
4. **实测数据驱动** — 先用 vision_analyze 采集客观数据，再解读
5. 🔴 **禁止盲信视觉模型的左右手判断** — 详见「左右手判断致命陷阱」

## 左右手判断致命陷阱 🔴

**视觉模型（vision_analyze）对左右手的判断系统性地不可靠。** 实测多次翻车——用户照片多为前置摄像头自拍，存在镜像翻转。模型看到的「拇指在左」可能实际在右。

### 镜像定律（2026-05-12 由多次翻车总结）

> **模型说「左手」→ 大概率实际是「右手」。模型说「右手」→ 大概率实际是「左手」。**

### 强制流程

1. vision_analyze 采集所有数据（纹线、手指、掌丘等）时**同时问拇指在哪一侧**
2. 出完整报告前**必须先问用户确认**：「视觉判为X手，请问实际是哪只手？」
3. 只拿用户确认后的结果做先天/后天框架
4. ❌ 禁止在自己不确认的情况下直接出左/右手报告

```
1. 数据采集 → vision_analyze(只问"是什么"，不问"代表什么")
   🔴 含拇指位置（判断左右手）→ 但必须向用户确认后再定先天/后天框架
   （视觉模型因前置摄像头镜像导致系统性左右手判断错误）
2. 五指分论 → 拇指=意志/土，食指=权力/木/巽宫，中指=自我/火/离宫，
               无名指=审美/金，小指=口才/水/兑宫
               ★ 必做：食指 vs 无名指长度对比
3. 三主纹精断 → 地纹(生命线)、人纹(智慧线)、天纹(感情线)
                逐纹分段解读（按年龄段）
4. 玉柱纹 + 太阳线 → 事业核心双线，按出发位置/深浅变化/终点判断
5. 八宫掌丘定位 → ★评级(★平坦 ★★中等 ★★★饱满)
6. 明堂分析 → 掌心痣、特殊标记（星纹/十字纹/岛纹等）
7. 综合雷达图 → 0-10 分量化 8 个维度
8. 事业/感情路径精准推荐 → ✅排名推荐 + ❌明确排除（具体到岗位级别）
9. 流年推运 → 玉柱纹分段对应年龄节点
```

## 雷达图八维度

| 维度 | 手相依据 |
|------|----------|
| 事业心 | 食指长度、巽宫饱满度 |
| 执行力 | 拇指长度/弯曲度、火星丘 |
| 创造力 | 智慧线走向（下垂型=创意）、坤宫 |
| 领导力 | 木星丘（巽宫）饱满度 |
| 社交力 | 金星丘（震宫）饱满度、指尖形状 |
| 表达力 | 水星丘（兑宫）饱满度、小指长度 |
| 抗压力 | 中指长度、土星丘（离宫） |
| 审美感 | 无名指、太阳丘（坤宫）、智慧线形态 |

## 十项禁止（Pitfalls）

1. ❌ 禁止泛泛说"适合创意工作" → 必须精确到岗位（如：品牌策略总监 / UX设计师 / 内容主编）
2. ❌ 禁止只说掌丘"饱满" → 必须★量化
3. ❌ 禁止用俗称 → 必须用古典术语
4. ❌ 禁止跳过食指vs无名指对比 → 这是事业vs感情的底层代码
5. ❌ 禁止跳过五指逐一分析
6. ❌ 禁止缩水报告 → 九步缺一不可
7. ❌ 禁止跳过量化雷达图
8. ❌ 禁止只说"中年运势好" → 必须用玉柱纹分段给具体年龄窗口
9. ❌ 禁止忽略明堂（掌心痣/特殊标记）
10. ❌ 禁止没有"明确排除"的不适合方向
11. 🔴 **禁止未经用户确认就使用视觉模型的左右手判断** → 前置摄像头自拍镜像导致模型系统性地判反。先确认再出报告。

## 感情专项额外要求

分析感情时必须加上：
- 妻妾纹（婚姻线）的数量/长短/深浅 → 精确到 27-32 岁年龄窗口
- 天纹终点精确到「食中指之间」vs「食指下方」vs「中指下方」
- 天纹形态三无水纹判断（无岛纹/无断裂/无链状）
- 无名指下是否有分叉线（感情升华线）
- 天地间距（天纹与人纹间距）= 心量大小
